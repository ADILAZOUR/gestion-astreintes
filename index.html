🔧 Comment créer le Flow Power Automate
1Ouvrir Power Automate → Créer → Flux instantané
2Choisir déclencheur : "Lors de la réception d'une requête HTTP"
3Copier le schéma JSON ci-dessous dans le champ "Corps JSON de la demande"
4Ajouter action : "Envoyer un e-mail (V2)" Outlook
5Champs : À → body/emailTo / CC → body/emailCC / Objet → body/subject / Corps → body/body
6Sauvegarder → Copier l'URL HTTP POST généré → Coller ci-dessus
